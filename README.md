## 使い方
1. このリポジトリを fork してね！
1. fork したリポジトリを、clone
1. このリポジトリも upstream とし登録しておいてね！
  ```bash
  git remote add upstream https://github.com/akeyace/learning-python-easy
  ```
1. fork したら自分のリポジトリで開発
1. 開発したらコミット、プッシュ
1. 自分のリポジトリから プルリクエストをこのリポジトリ宛に送ってね！
